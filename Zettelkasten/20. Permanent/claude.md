# 20. Permanent 폴더

## 목적과 역할
내 언어로 작성된 영구적인 지식 노트를 보관합니다. Zettelkasten의 핵심입니다.

## 들어가는 노트
- 하나의 아이디어/개념을 담은 원자적 노트
- 내 언어로 완전히 재작성된 지식
- 다른 노트와 연결 가능한 독립적 단위

## 작업 방식과 규칙
- 하나의 노트 = 하나의 아이디어 (원자성)
- 맥락 없이도 이해 가능하게 작성 (자기완결성)
- 최소 1개 이상의 다른 노트와 [[링크]] 연결
- 노트 제목: 명확한 주장/개념 (예: "복리는 시간이 핵심이다")
- 출처가 있다면 Literature/ 노트 참조 링크 추가
